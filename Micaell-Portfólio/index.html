Meu projeto: Projeto Corrida

Este projeto é uma página da web que abre um site chamado P5, que mostra como criar um jogo chamado Corrida. Para criar, é preciso do HTML, JS e CSS. Essas 3 tecnologias ajudam a criar o Corrida, onde os jogadores usam 4 emojis para criar seus corredores. Depois que o projeto é finalizado, é preciso enviar no curso Alura.
